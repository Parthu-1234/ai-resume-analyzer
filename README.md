## 🚀 Features

- **Python-based Resume Parsing** – Uses NLP techniques to extract candidate details.  
- **Keyword Matching** – Aligns resume content with job descriptions for better ATS performance.  
- **ATS Scoring System** – Provides a compatibility score to help candidates optimize resumes.  
- **Data Visualization** – Generates charts and tables to make analysis clear and actionable.  
- **Extensible Design** – Built with modular Python scripts, easy to expand with new features.
